Agora vamos reconstruir o vetor para que ele sirva como um fator de atualizao das sries em valores correntes. Como ns temos a informao da variao percentual em cada ms ser preciso adotarmos algumas etapas. Ns teremos que retirar da unidade percentual, dividindo todos os valores por 100 e em seguida somarmos 1, para que ao mutilicarmos os valores sejam apenas corrigidos de forma percentual. Agora, note que nosso vetor tem incio em 1997 e termina em 2016 e ns queremos que o primeiro valor seja capaz de atualizar nossa informaes de janeiro de 1997 at dezembro de 2016, ou seja, esse primeiro valor deve conter toda a inflao acumulada no perodo. Da mesma forma, o segundo valor do vetor deve trazer toda a inflao acumulada de fevereiro de 1997 at o final e assim sucessivamente, at o ponto em que o penltimo elemento deve atualizar apenas de novembro de 2016 para dezembro e o ltimo no deve alterar os dados, perodo de referncia.

Para obter algo assim vamos utilizar da funo cumprod() que nos retorna o produto de cada elemento do vetor com todos os elementos anteriores a ele (experimente ver como ela funciona fazendo cumprod(1:5)). Mas agora pense o seguinte, para que a gente consiga corrigir, por exemplo, o valor de outubro de 2016 at dezembro preciso que nosso antepenltimo elemento do vetor contenha apenas a inflao acumulada nos meses de outubro e novembro. Ou seja, dada a estrutura da funo cumprod() vai ser preciso criarmos o objeto ipca.acum invertendo nosso vetor IPCA e permitindo que os valores mais antigos possam ser corrigidos por toda a infao anterior. Na sequncia, contudo, ser preciso invertermos novamente ipca.acum, para que a correo que considera todo o perodo esteja na mesma linha do perodo inicial: janeiro de 1997. Nas duas inverses ns usaremos da seleo indicando a ordem que os elementos devem ser selecionados usando nrow(ipca):1.

ipca$IPCA <- ipca$IPCA/100 + 1

ipca$IPCA[nrow(ipca)] <- 1

ipca.acum <- cumprod(ipca$IPCA[nrow(ipca):1])

ipca$IPCA <- ipca.acum[nrow(ipca):1]

Repare que o fator de correo para o perodo inicial foi 3.503291, ou seja, em termos percentuais trata-se de uma taxa de 250,32% - utilize a calculadora de inflao do IBGE para conferir nossos resultados. Usando o Sistema Gerenciador de Sries Temporais do BCB

Das variveis usadas pelos autores ainda nos restam o PIB, o investimento pblico e a dvida pblica. Cada uma delas possui uma origem distinta, por isso, vamos comear com a dvida pblica para usarmos as ferramentas que nos possibilitam baixar dados diretamente do SGS do Banco Central do Brasil. Nesse caso no h um pacote como o sidrar do IBGE, necessrio buscar as informaes de forma mais direta. De todo modo, depois de feita a primeira vez voc pode salvar sua funo que importa dados e us-la toda vez que quiser.

Aqui ns vamos criar a funo bcbsgs que ter apenas dois argumentos: o nmero da srie a ser importada (serie) e o perodo para qual queremos as informaes (periodo). Enquanto a primeira informao ser dada com um nico nmero, a segunda ser um vetor com duas datas escritas da maneira como o R consegue interpret-las. Antes de criarmos nossa funo, contudo, ser preciso instalar um pacote responsvel pela leitura de dados no formato JSON, no qual o BCB exporta suas informaes.

install.packages("rjson")

Feito isso, vamos descrever os comandos de nossa funo bcbsgs. Em primeiro lugar ela ir definir o endereo em que o arquivo ser buscado no SGS, para isso far a fuso da srie que escolhermos no argumento serie com o restante do endereo padro do sistema. Na sequncia ela utilizar da funo rjson::fromJSON() para importar nossos dados, j usando como argumento a url que definimos acima. Como o objeto importado uma lista em que cada elemento composto por dois outros chamados $datae $valor, ns siplificamos o objeto aplicando a funo unlist() que transforma esses dados em um nico vetor no qual os elementos so nomeados como "data" e "valor". Assim, ns formamos nossa base de dados dividindo o vetor entre os dois tipos de informao e construindo um novo data.frame. A ltima etapa, por fim, consiste em ajustar o perodo de interesse, para o qu preciso primeiro atribuir a classe "Date" para nossa varivel data e na sequncia recortar a base de dados conforme os meses definidos em periodo.

bcbsgs <- function(serie, periodo = character()) {

# Definir srie 
url <- paste0("http://api.bcb.gov.br/dados/serie/bcdata.sgs.", 
              serie,
              "/dados?formato=json")

# Baixar arquivo
import <- unlist(rjson::fromJSON(file = url))

# Criar base de dados
dados <- data.frame(data = import[names(import) == names(import)[1]],
                    valor = as.numeric(import[names(import) == names(import)[2]]),
                    stringsAsFactors = F)

# Definir tipos das variveis
dados$data <- as.Date(dados$data, "%d/%m/%Y")
# Recortar data
dados <- dados[dados$data >= periodo[1] & dados$data <= periodo[2],]

return(dados)

}
